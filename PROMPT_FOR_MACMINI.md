Ты — AI-агент на Mac Mini. Твоя задача — применить D1-First миграцию к проекту evoWorkNew.

## Шаг 1: Прочитай документацию

Открой файл `D1_MIGRATION.md` в корне проекта. Там описана архитектура, все новые файлы и все изменения.

## Шаг 2: Примени ВСЕ изменения из ветки `ai`

Выполни:
```bash
git fetch origin
git checkout main
git merge origin/ai --no-edit
```

Если merge-конфликты → разреши их, приоритет у ветки `ai` (новый код).

## Шаг 3: Проверь TypeScript

```bash
cd packages/backend
npx tsc --noEmit
```

Должно быть 0 ошибок. Если есть ошибки → исправь, потом перепроверь.

## Шаг 4: Перезапусти сервер

```bash
# Убей старый процесс
pkill -f "tsx server" 2>/dev/null
sleep 2

# Запусти новый
cd /Users/.../evoWorkNew/packages/backend
nohup npx tsx server.ts > /tmp/evo-server.log 2>&1 &

# Подожди 45 секунд и проверь
sleep 45
curl http://localhost:3000/health
curl http://localhost:3000/api/shops
curl http://localhost:3000/api/employee/name-uuid
curl http://localhost:3000/api/evotor/sales-today
```

Все 4 curl должны вернуть корректный JSON.

## Шаг 5: Убедись, что cron запустился

```bash
tail -50 /tmp/evo-server.log
```

Должны быть строки:
- `[startup] Планы и зарплаты — готово`
- `Таблица 'shops' успешно создана или уже существует`
- `Таблица 'employees' успешно создана или уже существует`
- `Синхронизировано магазинов: N`

Если видишь ошибки — прочитай лог, исправь, перезапусти.

## Шаг 6: Открой фронтенд и проверь руками

1. Открой сайт в браузере
2. Убедись что страницы грузятся быстро (особенно «Финансовые отчёты» и «Зарплата»)
3. Проверь что данные совпадают с тем что было раньше

## Шаг 7: Доложи результат

Напиши мне коротко:
- Успешно ли применены изменения
- Сколько ошибок TypeScript (должно быть 0)
- Работает ли сервер
- Заметно ли ускорение
